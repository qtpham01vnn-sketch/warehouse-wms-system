# 🗺️ Workflow: Operation Map

Quy trình làm việc chuẩn cho Agent v2.

## 1. Input Analysis
- Phân tích yêu cầu từ người dùng.
- Xác định mục tiêu và phạm vi công việc.

## 2. Research & Planning
- Tìm kiếm thông tin liên quan (các tệp tin, tài liệu).
- Đề xuất kế hoạch hành động.

## 3. Implementation
- Thực hiện các thay đổi mã nguồn.
- Chạy các câu lệnh cần thiết.

## 4. Verification
- Kiểm tra tính đúng đắn của giải pháp.
- Đảm bảo tuân thủ các quy chuẩn đã đặt ra.
