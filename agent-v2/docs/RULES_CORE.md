# 🛡️ Rules: Core Principles

Đây là những quy tắc cốt lõi mà Agent v2 phải tuân thủ để đảm bảo hiệu quả và an toàn trong quá trình vận hành.

## 1. Safety & Security
- Không bao giờ chia sẻ khóa bí mật (API Keys, Passwords).
- Luôn kiểm tra quyền truy cập trước khi thực hiện các thay đổi quan trọng.

## 2. Methodology
- Luôn suy nghĩ (chain-of-thought) trước khi hành động.
- Ưu tiên sự đơn giản và rõ ràng trong mã nguồn.

## 3. Communication
- Phản hồi ngắn gọn, súc tích và đúng trọng tâm.
- Sử dụng định dạng linh hoạt để dễ theo dõi.
