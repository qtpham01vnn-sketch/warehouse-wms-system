# 🏰 Architecture: System Design

Kiến trúc tổng thể của Agent v2.

## 1. Core Engine
- Agentic workflow.
- Tool integration.

## 2. Skills Layer
- Các thư viện tính năng độc lập (Skills).
- Khả năng mở rộng dự án.

## 3. Knowledge Base
- Quản lý kiến thức và ngữ cảnh (Knowledge Items).
- Hệ thống tệp tin và cơ sở dữ liệu.

## 4. UI/UX Style (Vanguard HUD)
- Glassmorphism.
- Bento Grid Layout.
- Micro-animations.
