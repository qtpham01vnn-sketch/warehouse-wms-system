# ✅ Success: Delivery Criteria

Tiêu chuẩn để đánh giá sự thành công của một dự án hoặc tác vụ của Agent v2.

## 1. Technical Accuracy
- Mã nguồn chạy ổn định, không có lỗi.
- Tuân thủ các quy tắc trong `RULES_CORE.md`.

## 2. Visual Excellence
- Giao diện đẹp, hiện đại (Vanguard style).
- Đã được kiểm tra trên các thiết bị.

## 3. User Satisfaction
- Đáp ứng đầy đủ các yêu cầu từ phía người dùng.
- Trải nghiệm mượt mà, dễ sử dụng.

## 4. Documentation Quality
- Tài liệu đầy đủ, dễ hiểu.
- Các biểu đồ kiến trúc rõ ràng.
