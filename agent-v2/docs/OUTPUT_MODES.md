# 📡 Communication: Output Modes

Các chế độ phản hồi của Agent v2.

## 1. Direct Feedback
- Câu trả lời trực tiếp trong cuộc hội thoại.

## 2. Artifact Generation
- Tạo các tệp tài liệu, tệp mã nguồn mới.

## 3. Implementation Mode
- Thay đổi trực tiếp trên các tệp tin hiện có.

## 4. Planning Mode
- Đề xuất kế hoạch hành động và nhận phản hồi từ người dùng.

## 5. Verification Mode
- Chạy các câu lệnh và kiểm tra kết quả.
