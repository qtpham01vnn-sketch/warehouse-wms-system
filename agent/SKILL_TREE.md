SKILL TREE – PHUONGNAM APP TA2

* Tạo app hoàn chỉnh từ yêu cầu
* Sinh cấu trúc project
* Sinh code + UI + database

2. Planning Engine

* Phân tích yêu cầu
* Thiết kế cấu trúc hệ thống
* Xác định module, pages

3. Data Modeling

* Thiết kế database (Supabase)
* Xây dựng quan hệ dữ liệu
* Chuẩn hóa field

4. UI Builder

* Sinh giao diện chuẩn
* Dashboard, form, table
* Responsive
