# Skill: mega-prompt-generator
---
name: mega-prompt-generator
description: Generate full-scale structured mega prompt for product building
---

## When to use
- When user requests to create a new app/system/product blueprint
- When preparing input for AI app builder

## Input
- Product idea
- Target users (optional)
- Special requirements (optional)

## Steps
1. Identify product category
2. Expand business context
3. Define personas and JTBD
4. Generate features (MoSCoW)
5. Define tech stack
6. Define UI/UX system
7. Define page structure
8. Define data model
9. Define permissions
10. Define workflow
11. Generate build sequence
12. Assemble into mega prompt

## Output
- Full MEGA PROMPT ready for use in AI code generation
