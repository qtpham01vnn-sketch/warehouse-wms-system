# Skill: project-scaffolding
---
name: project-scaffolding
description: Generate project folder structure and base setup
---

## When to use
- When starting a new project
- When converting a prompt into structure

## Steps
1. Analyze project type
2. Create folder structure:
   - src/
   - pages/
   - components/
   - services/
   - types/
3. Define routing structure
4. Define base layout
5. Create placeholder files
6. Ensure scalability

## Output
- Folder structure
- File structure
- Initial architecture plan
