# ISO VERSION RULES

## Version Principles
- Mỗi lần upload file mới phải tạo version mới
- Chỉ có 1 version được phép Active
- Các version cũ phải chuyển sang Archived

## Required Fields
- version number
- change log
- uploaded by
- uploaded date
- status

## Version Status
- Active
- Archived

## Rules
- Không được mất lịch sử version cũ
- Không được có 2 version Active cùng lúc
- Có thể xem và tải version cũ nếu cần tra cứu
