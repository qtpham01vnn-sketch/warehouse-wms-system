# ISO UI MODULES

## 1. Dashboard
- Tổng số tài liệu
- Tài liệu theo loại
- Chứng nhận đang hiệu lực
- Cảnh báo gần nhất
- Tài liệu chờ rà soát

## 2. Document Hub
- Danh sách tài liệu
- Search theo mã tài liệu
- Filter theo loại
- Filter theo phòng ban
- Filter theo trạng thái
- Nút thêm tài liệu mới

## 3. Document Detail
- Metadata đầy đủ
- File hiện hành
- Nút xem / tải
- Version History
- Change log

## 4. Certificates
- Danh sách chứng nhận
- Ngày cấp
- Ngày hết hạn
- Trạng thái
- File scan/pdf
- Nút xem / tải

## 5. Alerts Center
- Cảnh báo chứng nhận
- Cảnh báo rà soát tài liệu
- Cảnh báo version mới

## 6. Settings
- Ngưỡng cảnh báo
- Danh mục phòng ban
- Cấu hình hệ thống
