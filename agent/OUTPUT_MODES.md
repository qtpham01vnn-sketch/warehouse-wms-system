OUTPUT MODES – PHUONGNAM APP TA2

1. APP
- Tạo ứng dụng hoàn chỉnh
- Bao gồm:
  + Cấu trúc project
  + UI
  + Database
  + Logic chính
- Ví dụ:
  + app-mmtb
  + app-kho
  + app-iso
  + app-vanphong

2. LANDING
- Tạo landing page hoàn chỉnh
- Bao gồm:
  + Hero section
  + Nội dung giới thiệu
  + CTA
  + Form / liên hệ
- Ví dụ:
  + landing-khoahoc
  + landing-congty
  + landing-sanpham

3. MEGA PROMPT
- Tạo prompt chuyên sâu
- Bao gồm:
  + Context
  + Goal
  + Rules
  + Output format
- Ví dụ:
  + prompt-iso
  + prompt-ban-hang
  + prompt-xay-app

4. FULL PACKAGE
- Tạo gói đầy đủ
- Bao gồm:
  + App
  + Landing
  + Prompt
  + README
  + Hướng dẫn triển khai

Nguyên tắc:
- Mỗi yêu cầu phải được phân loại vào 1 trong 4 output mode
- Không được xử lý khi chưa xác định output mode
- Output mode quyết định hướng build và skill cần gọi
